Postiz App – Social Media Content Manager

Postiz App is a public creator tool hosted at postizapp.julioaquino.cloud that helps TikTok users schedule, manage, and upload their short-form video content.

How the TikTok integration works

  • Users connect their TikTok account using the official TikTok Login Kit (OAuth).
  • After consenting, users can upload videos and metadata (caption, privacy, etc.) from Postiz App to TikTok via the TikTok Content Posting API.
  • Depending on the selected option, videos can be:
    • uploaded as drafts for further editing in TikTok Studio, or
    • published directly to the authenticated user’s own account.

User data & permissions

  • Postiz App only uses TikTok OAuth tokens and basic profile information to allow the user to post to their own TikTok account.
  • No automated bulk posting, scraping, or access to accounts without explicit user consent is performed.
  • Users can disconnect TikTok at any time from the integrations settings page, which stops further API access.
